# Horoscope adviser Backend
Small app, that can give you horoscope from different sources.

![alt text](https://img.shields.io/badge/python-3.10.12-orange)


## Technology stack

1. FastAPI
2. SQLAlchemy + Alembic
3. PostgreSQL
4. aiohttp
5. Docker, docker compose

## Development

#### Make lint, tests
```shell
make -C src lint
make -C src test
```
